## Web Programlama Ödevleri

### Projeyi Çalıştırma
##### 1. Npm paketlerini yükle
 ```bash
 npm install
 ```
 ##### 2. Projeyi çalıştır
 ```bash
 npm run dev
 ```
  ##### 3. Projeye tarayıcıda eriş `localhost:3000`
