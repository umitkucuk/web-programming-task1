## Web Programlama 1. Ödev

### Projeyi Çalıştırma
##### 1. Npm paketlerini yükle
 ```bash
 npm install
 ```
 ##### 2. Projeyi çalıştır
 ```bash
 npm run dev
 ```
  ##### 2. Projeye tarayıcıda eriş `localhost:3000`


### Advanced Özellikler:
 - Misafir seçebilme
 - Sağ altta `Şartlar, Mahremiyet, Para Birimi ve Daha Fazlası` butonuna tıklayarak footer açılıp kapatılabilir.
 - Infinite scroll
