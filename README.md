## Web Programlama 1. Ödev

### Projeyi Çalıştırma
##### Npm paketlerini yükle
 ```bash
 npm install
 ```
 ##### Projeyi Development Mod'da çalıştır
 ```bash
 npm run dev
 ```


### Advanced Özellikler:
 - Misafir seçebilme
 - Sağ altta `Şartlar, Mahremiyet, Para Birimi ve Daha Fazlası` butonuna tıklayarak footer açılıp kapatılabilir.
 - Infinite scroll
